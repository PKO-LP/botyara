# botyara
🎓 Бот расписания ГБПОУ ПК им. П.А. Овчинникова
Ваш персональный помощник для просмотра расписания занятий

🤔 Что умеет этот бот?
Бот создан специально для студентов и преподавателей колледжа, чтобы быстро и удобно получать актуальное расписание занятий.

📌 Основные возможности
Функция	Описание
📅 Просмотр расписания	Показывает расписание на любой выбранный день недели
🏢 Все корпуса	Поддерживает все корпуса колледжа
👥 Все группы	Работает со всеми учебными группами
⚡ Быстрая навигация	Удобное меню с кнопками, ничего не нужно вводить вручную
🔄 Автообновление	Всегда актуальные данные с официального сайта
📱 Простота	Работает на любом устройстве с Telegram
🏫 Доступные корпуса
Шенкурское отделение — все группы

Бибирево — 1, 2, 3, 4-5 курсы

Ярославский — 1, 2, 3, 4 курсы

Волоколамский и Свобода — объединенное расписание

👨‍🎓 Как пользоваться
Начать — напиши боту команду /start

Выбрать корпус — нажми на кнопку с нужным корпусом

Выбрать группу — выбери свою группу из списка

Выбрать день — нажми на нужный день недели

Готово! — получаешь расписание на выбранный день

🎯 Пример использования
text
Студент: /start
Бот: 👋 Добро пожаловать! Выберите корпус:
      [Шенкурское отделение] [Бибирево] [Ярославский] [Волоколамский и Свобода]

Студент: выбирает "Шенкурское отделение"
Бот: 🎓 Шенкурское отделение
      Выберите группу: [21ИС] [31ИС] [41ИС] ...

Студент: выбирает "21ИС"
Бот: 🎓 Группа: 21ИС
      Выберите день: [Пн] [Вт] [Ср] [Чт] [Пт] [Сб] [Вс]

Студент: выбирает "Пн"
Бот: 📅 Понедельник
      
      1. ⏰ 1 пара
         📚 Математика
         🏫 Каб. 213
      
      2. ⏰ 2 пара
         📚 Физика
         🏫 Каб. 205
⚙️ Технические детали
Бот берет расписание напрямую с официального сайта колледжа

Поддерживает форматы .xlsx и .xls

Кеширует файлы для быстрой загрузки

Работает 24/7 на любом сервере или компьютере

📦 Для разработчиков
Стек технологий:

Python 3.8+

python-telegram-bot

openpyxl (для .xlsx)

xlrd (для .xls)

requests

Установка:

bash
pip install python-telegram-bot requests openpyxl xlrd
Запуск:

bash
python bot.py
🔧 Решение проблем
Проблема	Решение
Бот не отвечает	Проверь интернет и перезапусти бота
Не загружается расписание	Проверь ссылки в конфиге или обнови кеш кнопкой "Обновить"
Ошибка с библиотеками	Установи все зависимости: pip install -r requirements.txt
